# RAGChatbot

A smart chatbot that helps you upload multiple PDFs, extract the text, and ask questions about the content — all in one place!

# 🛠️ Tech Stack

Python 3.10+

Streamlit
 – UI Framework

LangChain
 – LLM Orchestration

Google Gemini AI
 – Generative AI Models

FAISS
 – Vector Database for semantic search

PyPDF2
 – PDF Text Extraction
